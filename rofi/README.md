# Rofi

App launcher with wallust-synced theming. Launch with `Super + Space`.

## Themes

| Theme | Style |
|-------|-------|
| modern | List view, blur background, gradient selection |
| compact | 4x3 grid, large icons |
| minimal | Clean list view |
| floating | Centered floating window |
| macos9 | macOS 9 retro style |

Active theme is set in `config.rasi` (`@theme` line). All themes import colors from `themes/wallust-colors.rasi` (generated by wallust).

## Files

- `config.rasi` — main config
- `themes/*.rasi` — theme files
- `themes/wallust-colors.rasi` — wallust-generated colors (don't hand-edit)

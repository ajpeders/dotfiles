# Dotfiles — ~/.config

Arch Linux + Hyprland desktop dotfiles. Git repo in `~/.config/`, `install.sh` symlinks everything on a fresh machine.

## Stack

- **WM:** Hyprland — entry point `hypr/hyprland.conf`, sub-configs in `hypr/config/`
- **Bar:** Waybar
- **Launcher:** Rofi
- **Terminal:** Kitty
- **Shell:** Zsh (Oh My Zsh + Powerlevel10k) at `zsh/` via `ZDOTDIR`
- **Wallpaper:** awww (not swww/hyprpaper)
- **Theming:** wallust (Kanagawa-Wave) — generates colors for waybar, rofi, gtk, kitty, yazi
- **Notifications:** swaync

## Key Details

- `monitors.conf` is auto-generated by nwg-displays — don't hand-edit
- wallust-generated files (colors-waybar.css, wallust-colors.rasi, etc.) are live output — don't edit
- Named workspaces: dev(6), server(7), work(8), game(9), config, magic(scratchpad)
- `hyprpaper.conf` and `packages.txt` references to hyprpaper/waypaper are stale — awww is active
